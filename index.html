DRAM의 구조와 동작 원리

DRAM(Dynamic Random Access Memory)은 주기적으로 새로 고침을 해야 하는 휘발성 메모리입니다. DRAM은 다음과 같은 구조와 동작 원리를 가지고 있습니다:

  • 구조: DRAM 셀은 하나의 트랜지스터와 하나의 캐패시터로 구성됩니다. 이 두 가지 부품이 함께 DRAM 셀을 형성하며, 캐패시터는 데이터를 저장하고 트랜지스터는 데이터 접근을 제어합니다.
  • 동작 원리: DRAM 셀은 전하를 저장하여 비트 값을 나타냅니다. 이 저장된 전하는 시간이 지남에 따라 소모되므로, 데이터를 유지하기 위해 주기적으로 새로 고침(refresh)을 해야 합니다.

NAND 플래시 메모리의 구조와 동작 원리

NAND 플래시 메모리는 비휘발성 메모리로, 데이터가 전원이 꺼져도 유지됩니다. NAND 플래시 메모리는 다음과 같은 구조와 동작 원리를 가지고 있습니다:

  • 구조: NAND 플래시는 셀들이 직렬로 연결된 구조로, 셀은 트랜지스터와 플로팅 게이트로 구성됩니다. 데이터는 플로팅 게이트에 전하를 저장하여 기록됩니다.
  • 동작 원리: 데이터를 기록할 때, 플로팅 게이트에 전하를 주입하여 비트 값을 설정합니다. 데이터를 읽을 때는 플로팅 게이트의 전하 상태를 감지하여 데이터를 판별합니다. NAND 플래시는 블록 단위로 데이터를 삭제하고 새로 기록할 수 있습니다.
